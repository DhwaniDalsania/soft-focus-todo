# To-Do List App

This is a simple and aesthetic To-Do List application built using React.  
The app helps users manage daily tasks by separating active and completed items in a clean and organised interface.

## Features

- Add new tasks  
- Edit existing tasks  
- Mark tasks as completed  
- Delete tasks  
- Separate sections for active and completed tasks  
- Soft pastel UI with gradient background  

---

## Technologies Used

- React (useState hook)
- CSS / Tailwind CSS
- Material Symbols Icons

## Getting Started

### Clone the repository
```bash
git clone https://github.com/your-username/todo-app.git
```
### Install Dependecies
```bash
npm install
```

### Run the application
```bash
npm run dev
```

### Open the browser and visit:
```bash 
http://localhost:5173
```
---
## Application Logic

- The application uses React state to manage the current input value and the list of todos.
- Each todo item contains a task description and a completion status.
- Tasks are filtered into active and completed lists for better clarity and usability.
